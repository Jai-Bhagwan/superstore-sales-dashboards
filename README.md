# 📊 Sales Dashboard | Microsoft Excel

An interactive **Sales Dashboard** built in **Microsoft Excel** using Pivot Tables, Pivot Charts, Slicers, and Maps to analyze sales performance across categories, customers, states, and time periods. The dashboard enables dynamic filtering and provides valuable business insights for better decision-making.

---

# 🎯 Project Objective

The objective of this project is to transform raw sales data into an interactive dashboard that helps analyze sales performance, profit trends, customer contributions, and geographical distribution of sales. The dashboard allows users to filter data dynamically by **Category** and **Year** to gain meaningful business insights.

---

# 📂 Dataset Used

**Dataset:** Superstore Sales Dataset

The dataset contains transactional sales information including:

- Order ID
- Order Date
- Customer Name
- Product Category
- Product Name
- Sales
- Profit
- State
- Year
- Quantity
- Segment
- Region
- Ship Mode

**Total Records:** 9,994

---

# 📌 KPI Questions

The dashboard answers the following business questions:

1. Which product category generates the highest sales?
2. How has profit changed over the years?
3. Which states contribute the highest sales?
4. Which customers generate the highest profit?
5. What is the monthly sales trend?
6. How many customers placed orders each year?
7. How do sales and profits vary across different categories?
8. How does business performance change after applying filters?

- Year

# 📊 Dashboard Insights

### 📌 Sales by Category

- Phones generated the highest sales.
- Chairs and Storage are also strong-performing categories.
- Supplies and Fasteners contribute the least sales.

### 📌 Profit Trend

- Technology consistently generated the highest profit.
- Office Supplies showed steady growth.
- Furniture remained comparatively lower in profitability.

### 📌 State-wise Sales

- California contributes the largest share of total sales.
- Texas and New York are among the top-performing states.

### 📌 Top Customers

The dashboard identifies the Top 5 customers contributing the highest profits, helping businesses understand valuable customer segments.

### 📌 Monthly Sales

Sales increase significantly during the last quarter of the year, indicating strong seasonal demand.

### 📌 Customer Count

Customer orders remain relatively consistent, with slight variation across years.

# 💡 Business Recommendations

- Focus marketing efforts on high-performing categories like Phones and Technology.
- Strengthen sales strategies in lower-performing states.
- Improve inventory planning for months with peak sales.
- Build loyalty programs for top customers.
- Increase promotions for low-performing product categories.
